# LoL Ghost

Create with Electron / Socket.io / React

## Installation
Install all dependencies:

    yarn

## Commands
Start developement :

    yarn electron-dev
Build for all platforms

    yarn electron-pack-all
Build for OSX

    yarn electron-pack-osx
Build for Windows

    yarn electron-pack-win
